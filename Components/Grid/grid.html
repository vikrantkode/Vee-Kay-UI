Simplified Grid

Grid is a two-dimensional grid system used to work on the layout of UI elements and segments of a webpage. The Grid comprises horizontal and vertical lines to form rows and columns, much like a table.

Grid - 2

Grid 2 is a two colum grid in which a whole grid is divided into two colums by using property `grid-template-columns: repeat(2, 1fr);`.

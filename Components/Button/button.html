Button

Buttons allows user to take actions and make choices with a single tap. Buttons communicate actions that users can take. They are typically placed throughout the website UI.

Primary Button

Primary buttons generally controls the actions of the UI.

Primary Outline Button

Primary outlines buttons are just the other varient of primary buttons where class name is btn-default_outline.

Link Button

Link buttons are the buttons which generally used to navigate to the place where you want to.

Icon Button

Icon buttons are the buttons which generally used to navigate to parent button. pass class name btn-icon to implement it.

Float Button

Float Buttons are buttons which generally appears at the bottom left or right of the UI.

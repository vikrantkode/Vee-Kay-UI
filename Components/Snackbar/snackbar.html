Snackbar

Snackbar/Toast is a type of interaction happened between user and UI and this snackbar stays on the screen for 1-2 seconds.

Baseline Snackbar

Baseline Snackbar is usually shown in the middle of the screen at the bottom. It is generally used to show pop-up for warning, success, errors etc.

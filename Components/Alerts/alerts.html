Alert

Alerts can be used to show an important message to the user so that it can grab user's attention without any disturbance.

Simple Alert

Simple Alerts are the alerts which can be created by giving class name as alert.

Error Alert..! Check it now.!
Success Alert..! Check it now.!
Warning Alert..! Check it now.!
Primary Alert..! Check it now.!

Outline Alert

Outline Alerts are the alerts which can be created by giving class name as alert followed by its varient.

Error Alert..! Check it now.!
Success Alert..! Check it now.!
Warning Alert..! Check it now.!
Primary Alert..! Check it now.!

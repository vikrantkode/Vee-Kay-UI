Avatar

The Avatar component is used to represent a user, displays the profile picture, initials or fallback icon.

Rounded Avatar

Vee-Kay UI has three rounded shaped avator which includes avator xl, avator and avator sm. Size of Rounded Avatar is created by passing class="avator_round" with xl, sm.

Square Avatar

Vee-Kay UI has three squared shaped avator which includes avator xl, avator and avator sm. Size of Square Avatar is created by passing class="avator_square" with xl and sm.
